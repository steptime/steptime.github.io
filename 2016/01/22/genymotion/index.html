介绍

Genymotion是一套完整的工具，它提供了Android虚拟环境。它简直就是开发者、测试人员、推销者甚至是游戏玩家的福音。Genymotion支持Windows、Linux和Mac OS等操作系统，容易安装和使用。号称最快的安卓模拟器,确实比android自带的模拟器快多了。

环境要求

操作系统要求
需要满足以下操作系统之一:

  • Microsoft Windows XP SP3 (32 or 64 bits)
  • Microsoft Windows Vista (32 or 64 bits)
  • Microsoft Windows 7 (32 or 64 bits)
  • Linux Ubuntu 12.04
  • Linux Ubuntu 12.10
  • Mac OS X 10.X

应用程序要求

Oracle VirtualBox >= 4.1（版本越高越好）
